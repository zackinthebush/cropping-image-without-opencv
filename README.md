# Image Cropping Without OpenCV

A Python script that crops images by 32x32 pixel blocks without using OpenCV.

## Technologies
- **Language**: Python

## Purpose
Implement image cropping functionality using pure Python, demonstrating image manipulation without external computer vision libraries.

## Features
- 32x32 pixel block cropping
- No external library dependencies (besides basic Python)
- Image processing from scratch
- Lightweight implementation

## Usage
Run the Python script to crop images into 32x32 blocks without OpenCV dependency.
